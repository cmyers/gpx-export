API Reference

gpx-export exposes a focused, typed API for building GPX 1.1 documents. The main export is generateGpx(track, options?).

generateGpx(track, options?)

Creates GPX 1.1 XML with metadata, track name, segment, and points. Garmin TrackPointExtension output is automatically added when speed values are present. 

import { generateGpx } from "gpx-export";

const xml = generateGpx(
  {
    name: "Morning Ride",
    points: [
      { lat: 54.57, lon: -1.31, time: new Date(), speed: 5.2 }
    ]
  },
  { creator: "my-app" }
);

Returns

  • string - GPX 1.1 XML document

GpxPoint

Defines each point in the generated track. 

interface GpxPoint {
  lat: number;
  lon: number;
  time: Date;
  speed?: number;     // m/s
  elevation?: number; // metres
}

GpxTrack

Defines metadata and ordered points. 

interface GpxTrack {
  name: string;
  createdAt?: Date;
  points: GpxPoint[];
}

GpxOptions

interface GpxOptions {
  creator?: string;
}

Behavior notes

  • Creator defaults to gpx-export.
  • Track name and creator are XML-escaped before output.
  • Metadata time defaults to first point time, then current time.
  • Elevation is formatted to 2 decimal places.
  • Speed is formatted to 4 decimal places as gpxtpx:speed.
